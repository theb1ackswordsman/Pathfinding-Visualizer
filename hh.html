Pathfinding Visualizer

Data Flow Diagram and System Architecture

System Components

Grid Management

  • Creates 20x30 grid structure
  • Handles user interactions
  • Manages cell states (start, end, wall)
  • Generates random mazes

Algorithm Engine

  • Implements BFS algorithm
  • Implements DFS algorithm
  • Implements Dijkstra's algorithm
  • Implements A* search algorithm

Visualization Renderer

  • Color codes different node types
  • Animates algorithm progression
  • Updates DOM in real-time
  • Handles reset functionality

Data Flow Process

1
User initializes the grid or generates a random maze
↓
2
Grid Management Module creates the grid structure and stores cell states
↓
3
User selects an algorithm (BFS, DFS, Dijkstra, or A*) to run
↓
4
Algorithm Engine processes the grid data and computes the path
↓
5
Visualization Renderer receives algorithm steps and updates the UI
↓
6
Statistics (time complexity, space used, runtime) are calculated and displayed
↓
7
When path is found, it's highlighted with animation
